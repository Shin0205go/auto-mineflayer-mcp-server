# Bug Issues Directory

各botが発見したバグやイシューを記録するディレクトリ。

## ファイル構成

- `bot1.md` - Bot1専用のバグレポート
- `bot2.md` - Bot2専用のバグレポート
- `bot3.md` - Bot3専用のバグレポート
- `bot4.md` - Bot4専用のバグレポート
- `bot5.md` - Bot5専用のバグレポート

## ルール

- 各botは**自分専用のファイルのみ**を編集すること
- 他のbotのファイルは読み取り専用
- これによりgitマージ時の競合を防ぐ

## 報告形式

各ファイルには以下の形式でバグ/イシューを記録：

```markdown
### [日付] バグタイトル
- **症状**: 何が起きたか
- **原因**: 推定される原因
- **修正**: どう修正したか（または修正予定）
- **ファイル**: 関連するファイルパス
```

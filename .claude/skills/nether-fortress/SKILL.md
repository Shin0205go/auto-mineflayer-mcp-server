# ネザー要塞探索スキル

ブレイズロッドとウィザースケルトンの頭蓋骨を集める。

## ネザー要塞の特徴

- **ネザーレンガ**で構成された巨大構造物
- **Y=30-70**付近に生成
- **南北方向（X軸）**に沿って生成されやすい

## 見つけ方

### 重要: ±300ブロック以内に要塞がない場合
- x=-300〜+300, z=-300〜+300で見つからない場合は、**±400〜800ブロック**を探す
- ネザー座標400=overworld座標3200

### 1. X軸方向に段階的移動（チャンクロード必須）
```
# ステップ1: 100ブロック移動してネザーレンガを探す
mc_navigate(x=current_x + 100, y=60, z=current_z)
mc_navigate(target_block="nether_bricks", max_distance=200)

# 見つからなければ次の100ブロックへ
mc_navigate(x=current_x + 200, y=60, z=current_z)
mc_navigate(target_block="nether_bricks", max_distance=200)
# ... x=500まで繰り返す
```
- bot.findBlocksはチャンクキャッシュ内しか探せない → 移動してからチャンクをロードする
- 要塞は**X軸方向に並ぶ傾向**
- Z軸よりX軸方向の探索が効率的

### 2. 高度を調整
- **Y=60-70付近**を歩く（橋や通路が見える）
- 溶岩湖の上を飛び越えられる足場を作りながら

### 3. 目印
- **ネザーレンガ**の構造物
- **ブレイズ**の光（遠くからでも見える）
- **ウィザースケルトン**の影

## ブレイズ攻略

### 特徴
- 火の玉を3連射
- 近接攻撃も可能
- 水をかけると大ダメージ（ネザーでは使えない）

### 攻略法
```
1. 食料を十分に持ってから（推奨: cooked_beef x16以上）
2. HP確認: mc_status()
3. mc_combat(target="blaze", flee_at_hp=10) で攻撃
   - flee_at_hp=10 必須 (Blazeの火の玉は3ダメ×3連射)
4. HP低下時は mc_eat() で回復してから再戦
5. 7体倒したらポータルに戻る
```

### 注意
- **Blazeは空中を浮く** - 近づいて攻撃するとbotが地上から攻撃できる
- **fleeしたら必ず食料を食べてから再戦**
- **Food=0でネザーを歩くな** - HP回復しないまま死ぬ

### 必要数
- **ブレイズロッド 6本以上**（エンダーアイ12個分）
- 推奨: 10本以上（予備）

### ブレイズパウダー
```
mc_craft(item="blaze_powder")
# ブレイズロッド1本 → ブレイズパウダー2個
```

## ウィザースケルトン

### 特徴
- 黒い骨格、石の剣装備
- 攻撃で**衰弱**状態付与
- **頭蓋骨**をドロップ（低確率）

### 頭蓋骨集め
- ドロップ率: **2.5%**（ドロップ増加IIIで5.5%）
- **ウィザー召喚**に3個必要

### 攻略法
- 高さ2ブロックの通路に誘い込む
- ウィザスケは高さ2.4なので入れない
- 安全に攻撃可能

## ネザーウォート

### 場所
- 要塞の**階段部屋**
- ソウルサンドの上に生えている

### 用途
- **ポーション醸造**の基本材料
- 必ず回収しておく

## 危険なモブ

| モブ | 対策 |
|------|------|
| ブレイズ | 盾で防御、近接攻撃 |
| ウィザスケ | 距離を取る、狭い通路に誘導 |
| マグマキューブ | 弓で攻撃、分裂に注意 |
| ガスト | 火球を打ち返す |

## 帰還方法

1. **ポータルの座標を記録**
   `mc_chat(message="[拠点] ネザーポータル: x=..., y=..., z=...")`

2. **迷ったら**
   - 丸石で道を作りながら戻る
   - コンパスは使えない（ネザーでは機能しない）
   - **ロードストーン**があれば使用可能

# ネザー要塞探索スキル

ブレイズロッドとウィザースケルトンの頭蓋骨を集める。

## ネザー要塞の特徴

- **ネザーレンガ**で構成された巨大構造物
- **Y=30-70**付近に生成
- **南北方向（X軸）**に沿って生成されやすい

## 見つけ方

### 1. X軸方向に移動
```
minecraft_move_to { x: current_x + 100, y: 60, z: current_z }
```
- 要塞は**X軸方向に並ぶ傾向**
- Z軸よりX軸方向の探索が効率的

### 2. 高度を調整
- **Y=70付近**を歩く（橋や通路が見える）
- 溶岩湖の上を飛び越えられる足場を作りながら

### 3. 目印
- **ネザーレンガ**の構造物
- **ブレイズ**の光（遠くからでも見える）
- **ウィザースケルトン**の影

## ブレイズ攻略

### 特徴
- 火の玉を3連射
- 近接攻撃も可能
- 水をかけると大ダメージ（ネザーでは使えない）

### 攻略法
```
1. 盾で火の玉を防ぐ
2. 攻撃の合間に近づく
3. 剣で攻撃
4. 後退して盾構え
5. 繰り返し
```

### 必要数
- **ブレイズロッド 6本以上**（エンダーアイ12個分）
- 推奨: 10本以上（予備）

### ブレイズパウダー
```
minecraft_craft { item_name: "blaze_powder" }
# ブレイズロッド1本 → ブレイズパウダー2個
```

## ウィザースケルトン

### 特徴
- 黒い骨格、石の剣装備
- 攻撃で**衰弱**状態付与
- **頭蓋骨**をドロップ（低確率）

### 頭蓋骨集め
- ドロップ率: **2.5%**（ドロップ増加IIIで5.5%）
- **ウィザー召喚**に3個必要

### 攻略法
- 高さ2ブロックの通路に誘い込む
- ウィザスケは高さ2.4なので入れない
- 安全に攻撃可能

## ネザーウォート

### 場所
- 要塞の**階段部屋**
- ソウルサンドの上に生えている

### 用途
- **ポーション醸造**の基本材料
- 必ず回収しておく

## 危険なモブ

| モブ | 対策 |
|------|------|
| ブレイズ | 盾で防御、近接攻撃 |
| ウィザスケ | 距離を取る、狭い通路に誘導 |
| マグマキューブ | 弓で攻撃、分裂に注意 |
| ガスト | 火球を打ち返す |

## 持ち物チェックリスト

### 必須
- [ ] ダイヤ/ネザライト防具
- [ ] ダイヤ剣（鋭さ付き推奨）
- [ ] 盾
- [ ] 弓と矢64本以上
- [ ] 食料64個以上
- [ ] 丸石128個以上（足場用）
- [ ] 火打石と打ち金（ポータル再点火用）

### 推奨
- [ ] 金装備1個（ピグリン対策）
- [ ] エンダーチェスト
- [ ] 治癒のポーション
- [ ] 耐火のポーション

## 帰還方法

1. **ポータルの座標を記録**
   ```
   remember_location { name: "nether_portal", ... }
   ```

2. **迷ったら**
   - 丸石で道を作りながら戻る
   - コンパスは使えない（ネザーでは機能しない）
   - **ロードストーン**があれば使用可能

# ネザーゲート作成スキル

ネザーポータルはエンダードラゴン討伐に必須。ブレイズロッドとエンダーパールを集める。

## 必要素材

| アイテム | 数量 | 入手方法 |
|---------|------|---------|
| 黒曜石 (obsidian) | 10個（最小） | 溶岩+水、またはダイヤピッケルで採掘 |
| 火打石と打ち金 | 1個 | 鉄インゴット1 + 火打石1 |

## 前提条件

- **ダイヤのピッケル** または **溶岩+水バケツ**
- 鉄インゴット（火打石用）
- 砂利から火打石を入手

## 方法1: 溶岩と水で作る（ダイヤピッケル不要）

### 手順

1. **溶岩溜まりを見つける**
   ```
   minecraft_find_block { block_name: "lava", max_distance: 64 }
   ```

2. **水バケツを用意**
   ```
   minecraft_craft { item_name: "bucket" }  # 鉄インゴット3個
   # 水源で水を汲む
   minecraft_use_item {}  # バケツを持って水に向かって使用
   ```

3. **ポータル枠を作る**
   - 溶岩源に水を流す → 黒曜石に変化
   - 4x5の枠を形成（角は不要）

   ```
   # 枠の形（O=黒曜石, .=空気）
   .OOO.
   O...O
   O...O
   O...O
   .OOO.
   ```

## 方法2: 黒曜石を採掘（ダイヤピッケル必要）

1. **溶岩を黒曜石にする**
   ```
   # 溶岩源に水を流す
   minecraft_place_block { block_type: "water_bucket", ... }
   ```

2. **ダイヤピッケルで採掘**
   ```
   minecraft_equip_item { item_name: "diamond_pickaxe" }
   minecraft_dig_block { x, y, z }  # 黒曜石1個につき約10秒
   ```

3. **10個集めてポータル設置**

## ポータル起動

1. **火打石と打ち金を作成**
   ```
   minecraft_craft { item_name: "flint_and_steel" }
   ```

2. **ポータル内部で使用**
   ```
   minecraft_use_item {}  # ポータル枠の内側で
   ```

3. **紫のポータルが出現したら成功**

## ネザーに入る前の準備

### 必須アイテム
- [ ] 鉄/ダイヤ装備一式
- [ ] 剣（鉄以上）
- [ ] 弓と矢（64本以上）
- [ ] 食料（焼き肉64個以上）
- [ ] 丸石（64個以上、足場用）
- [ ] 火打石と打ち金（予備）

### 推奨アイテム
- [ ] 金インゴット（ピグリン対策）
- [ ] エンダーチェスト
- [ ] 盾

## ネザー要塞の見つけ方

1. **X軸方向に移動**（要塞はX軸に沿って生成）
2. **Y=70付近を探索**（要塞の高さ）
3. **ネザーレンガを目印に**

## 危険なモブ

| モブ | 対策 |
|-----|------|
| ガスト | 弓で撃つ、火球を打ち返す |
| ブレイズ | 盾で防御、近接攻撃 |
| ピグリン | 金装備で中立化 |
| マグマキューブ | 距離を取って弓 |

## ブレイズロッド収集

```
minecraft_fight { entity_name: "blaze" }
```

- ネザー要塞のブレイズスポナー付近
- **必要数**: 最低6本（エンダーアイ12個分）
- 推奨: 10本以上

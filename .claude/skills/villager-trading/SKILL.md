## 司書ガチャ

エンチャント本の内容は村人ごとにランダム。

### 手順
```
1. 書見台を設置
2. 無職村人が司書になる
3. 取引内容を確認
4. 欲しくなければ書見台を壊す
5. 再設置（取引内容がリセット）
6. 目的の本が出るまで繰り返し
```

**注意**: 一度でも取引すると固定される

### 目標
- **修繕の本**: 10-20エメラルド程度で出たら確保
- **効率強化V**: 高レベルエンチャント
- **耐久力III**: 汎用性高い

## 村人の増やし方

### 条件
1. **ベッド**が余っている（村人数+1以上）
2. **食料**を持っている（パン3個以上）
3. **2人以上**の村人がいる

### 繁殖手順
```
1. ベッドを多めに設置
2. 村人にパンを投げる（最低6個）
   minecraft_drop_item { item_name: "bread", count: 6 }
3. 自動で繁殖開始（ハートが出る）
4. 子供村人が生まれる
5. 成長まで20分
```

## 取引ホール

複数の村人を管理する施設。

### 基本設計
```
# 横から見た図
[村人][職業ブロック]|[村人][職業ブロック]|...
    ↓     通路      ↓
  [ベッド]        [ベッド]
```

### ポイント
- 村人は**1x1**のスペースに閉じ込める
- 職業ブロックは**村人の手が届く位置**に
- ベッドも近くに配置（夜に寝る）
- **松明**で明るく（ゾンビ襲撃防止）

## ゾンビ治療

ゾンビ村人を治療すると**大幅割引**。

### 手順
```
1. ゾンビ村人を確保（閉じ込める）
2. 弱化のスプラッシュポーションを投げる
3. 金のリンゴを与える（使用）
4. 3-5分で治療完了
```

### 効果
- 治療後の村人は**永久割引**
- 1エメラルド=1アイテムまで下がることも

## 襲撃イベント対策

### 発生条件
- 不吉な予感（Bad Omen）を持って村に入る

### 対策
- 牛乳を飲んでBad Omen解除
- または襲撃を撃退（報酬あり）

### 報酬（Hero of the Village）
- 全村人が大幅割引（一時的）

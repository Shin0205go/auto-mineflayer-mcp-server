# エンチャントスキル

装備を強化するエンチャントシステムの攻略。

## エンチャントテーブルの作成

### 必要素材
| アイテム | 数量 | 入手方法 |
|---------|------|---------|
| ダイヤモンド | 2個 | Y=-59で採掘 |
| 黒曜石 | 4個 | 溶岩+水→ダイヤピッケル |
| 本 | 1冊 | 革1+紙3 |

```
minecraft_craft { item_name: "enchanting_table" }
```

## 本棚の設置（最重要）

### 最大レベル(30)に必要
- **本棚15個**が必要
- テーブルから**1ブロック空けて**配置
- **高さ2段まで**有効

### 配置図
```
# 上から見た図（E=エンチャントテーブル、B=本棚、.=空気）
B B B B B
B . . . B
B . E . B
B . . . B
B B B B B

# 横から見た図
B B B B B
B . . . B  ← 2段目の本棚
. . E . .  ← 床（Eの周り1マス空ける）
```

### 本棚の作成
```
minecraft_craft { item_name: "bookshelf" }
# 本3冊 + 板材6個
```

**本の作成**:
```
minecraft_craft { item_name: "book" }
# 革1 + 紙3
```

**紙の作成**:
```
minecraft_craft { item_name: "paper" }
# サトウキビ3
```

## 経験値の稼ぎ方

### 序盤
- モブを倒す（ゾンビ、スケルトン）
- 鉱石を精錬（鉄、金）
- 取引（村人）

### 中盤以降
- **モブトラップ**（別スキル参照）
- **エンダーマントラップ**
- **ブレイズトラップ**

### 経験値効率
| 方法 | XP/時間 |
|------|--------|
| 通常狩り | 100-200 |
| モブトラップ | 500-1000 |
| エンダーマントラップ | 2000+ |

## 金床（修理・合成）

### 作成
```
minecraft_craft { item_name: "anvil" }
# 鉄ブロック3 + 鉄インゴット4
```

### 用途
1. **修理**: 同じアイテム2個を合成
2. **エンチャント本適用**: 本のエンチャントを付与
3. **名前変更**: アイテムに名前をつける

### コスト
- 合成コストは累積（何度も修理すると高くなる）
- **40レベル超え**は金床で操作不可

## エンチャント本

### 入手方法
- 村人取引（司書）
- ダンジョンチェスト
- 釣り

### 司書ガチャ
1. 司書の村人を用意
2. 職業ブロック（書見台）を壊す
3. 置き直す
4. 欲しいエンチャントが出るまで繰り返し

## Tips

- **レベル30エンチャント**を狙う（最も効率的）
- 外れたら**砥石**でエンチャント除去（経験値少量回収）
- **修繕**は釣りか司書取引で入手
- ダイヤ装備より**ネザライト装備**が最終目標

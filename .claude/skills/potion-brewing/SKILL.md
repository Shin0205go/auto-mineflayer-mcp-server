# ポーション醸造スキル

強力なバフと攻撃ポーションの作り方。

## 醸造台の作成

### 必要素材
| アイテム | 数量 | 入手方法 |
|---------|------|---------|
| ブレイズロッド | 1本 | ネザー要塞 |
| 丸石 | 3個 | 採掘 |

```
minecraft_craft { item_name: "brewing_stand" }
```

## 基本材料

### 必須
| アイテム | 用途 | 入手方法 |
|---------|------|---------|
| ネザーウォート | 奇妙なポーション | ネザー要塞 |
| ブレイズパウダー | 燃料 | ブレイズロッド |
| ガラス瓶 | 容器 | ガラス3個 |
| 水 | ベース | 水源で汲む |

### 効果材料
| アイテム | 効果 |
|---------|------|
| 金のニンジン | 暗視 |
| マグマクリーム | 耐火 |
| ガストの涙 | 再生 |
| きらめくスイカ | 即時回復 |
| 砂糖 | 俊敏 |
| ウサギの足 | 跳躍 |
| ファントムの皮膜 | 低速落下 |
| フグ | 水中呼吸 |
| クモの目 | 毒 |
| 発酵したクモの目 | 効果反転 |

### 修飾材料
| アイテム | 効果 |
|---------|------|
| レッドストーン | 効果時間延長 |
| グロウストーン | 効果強化 |
| 火薬 | スプラッシュ化 |
| ドラゴンブレス | 残留ポーション化 |

## 醸造の基本

### 手順
```
1. 水入り瓶を3つセット
2. ブレイズパウダーを燃料に
3. ネザーウォート投入 → 奇妙なポーション
4. 効果材料を投入 → 目的のポーション
5. (オプション) レッドストーン/グロウストーンで強化
6. (オプション) 火薬でスプラッシュ化
```

### 醸造チャート
```
水入り瓶
    ↓ ネザーウォート
奇妙なポーション
    ↓ 効果材料
各種ポーション
    ↓ レッドストーン（延長）or グロウストーン（強化）
強化ポーション
    ↓ 火薬
スプラッシュポーション
    ↓ ドラゴンブレス
残留ポーション
```

## 重要ポーションレシピ

### 戦闘用
| ポーション | 材料 | 効果 |
|-----------|------|------|
| 治癒 | きらめくスイカ | HP即時回復 |
| 再生 | ガストの涙 | HPゆっくり回復 |
| 力 | ブレイズパウダー | 攻撃力UP |
| 耐火 | マグマクリーム | 炎・溶岩無効 |

### 探索用
| ポーション | 材料 | 効果 |
|-----------|------|------|
| 暗視 | 金のニンジン | 暗闘でも見える |
| 水中呼吸 | フグ | 水中で呼吸可能 |
| 俊敏 | 砂糖 | 移動速度UP |
| 低速落下 | ファントムの皮膜 | ゆっくり落下 |

### 攻撃用（スプラッシュ）
| ポーション | 材料 | 効果 |
|-----------|------|------|
| 毒 | クモの目 | 継続ダメージ |
| 鈍化 | 発酵クモの目+俊敏 | 移動速度低下 |
| 弱化 | 発酵クモの目 | 攻撃力低下 |
| ダメージ | 発酵クモの目+治癒 | 即時ダメージ |

## おすすめポーション

### エンドラ戦
1. **低速落下** - 吹き飛ばし対策
2. **治癒II** - 緊急回復
3. **再生II** - 継続回復
4. **力II** - 攻撃力強化

### ネザー探索
1. **耐火（8分）** - 溶岩対策
2. **治癒II** - 緊急回復

### 水中探索
1. **水中呼吸（8分）** - 必須
2. **暗視（8分）** - 視界確保

## 効果時間

### 通常 vs 延長 vs 強化
| タイプ | 時間 | 効果 |
|--------|------|------|
| 通常 | 3分 | 標準 |
| 延長 | 8分 | レッドストーン |
| 強化 | 1分30秒 | グロウストーン |

**注意**: 延長と強化は同時に付けられない

## ポーション保管

### チェストに常備
```
# 冒険用セット
- 治癒II x8
- 再生 x4
- 耐火（8分） x4
- 暗視（8分） x4
- 低速落下 x4
```

### 醸造台の近くに
- ネザーウォート（栽培推奨）
- ブレイズパウダー（大量）
- 各種材料のチェスト

| 治癒 | → | ダメージ |

| 俊敏 | → | 鈍化 |

| 跳躍 | → | 鈍化 |

## 自動醸造機

### 基本構造
```
[ホッパー] → [醸造台] → [ホッパー] → [チェスト]
     ↑
[チェスト]（材料）
```

### 仕組み
1. 上のホッパーから材料投入
2. 醸造完了後、下のホッパーで回収
3. コンパレーターで完了検知

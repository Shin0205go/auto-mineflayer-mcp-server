# エンダードラゴン討伐スキル

Minecraftの最終ボス、エンダードラゴンを倒す方法。

## 前提条件

### 必須アイテム
| アイテム | 数量 | 用途 |
|---------|------|------|
| エンダーアイ | 12個以上 | ポータル起動 |
| ダイヤ/ネザライト防具 | フル | 防御 |
| ダイヤ/ネザライト剣 | 1本 | 攻撃 |
| 弓（パワー付き） | 1本 | クリスタル破壊 |
| 矢 | 128本以上 | 弓用 |
| 食料 | 64個以上 | 回復 |
| 丸石/土 | 128個以上 | 足場・塔登り |
| エンダーパール | 16個以上 | 緊急脱出 |
| 水バケツ | 2個 | 落下軽減 |
| ベッド | 1個 | リスポーン設定 |

### 推奨エンチャント
| 装備 | エンチャント |
|------|-------------|
| 剣 | 鋭さV、ノックバック |
| 弓 | パワーV、無限 |
| 防具 | 防護IV、落下軽減IV |
| ブーツ | 落下軽減IV |

## エンドポータルを見つける

### 1. エンダーアイを投げる
```
minecraft_use_item {}  # エンダーアイを持って
```
- アイが飛んでいく方向に要塞がある
- 距離が近いと下に向かう

### 2. 要塞を掘り当てる
- 地下に生成（Y=0-40付近）
- 石レンガの構造物を探す

### 3. ポータル部屋を見つける
- 溶岩プールの上にポータルフレーム
- シルバーフィッシュスポナーがある

### 4. ポータルを起動
- 12個のフレームにエンダーアイをはめる
- 黒いポータルが出現したら完成

## ジ・エンドへの準備

### 直前チェック
- [ ] 装備を全て修理
- [ ] エンダーチェストに予備装備
- [ ] ベッドでリスポーン設定
- [ ] インベントリ整理

**重要**: ジ・エンドに入ったら倒すまで戻れない！

## 戦闘フェーズ

### フェーズ1: エンドクリスタル破壊

**最優先目標**: 黒曜石の塔の上のクリスタルを全て破壊

```
# クリスタルがあるとドラゴンが回復し続ける
# 弓で撃つ
minecraft_use_item {}  # 弓を構えてクリスタルを狙う
```

#### 鉄格子付きクリスタル
- 弓が通らない
- 塔に登って直接破壊

```
# 塔の登り方
1. 丸石で螺旋階段を作る
2. またはエンダーパールで飛ぶ
3. 鉄格子を壊してクリスタル破壊
4. 水バケツで降りる
```

### フェーズ2: ドラゴン攻撃

#### ドラゴンの行動パターン
1. **旋回**: 塔の周りを飛ぶ（弓で攻撃）
2. **突進**: プレイヤーに向かって飛ぶ（剣で攻撃）
3. **ブレス**: 中央の噴水で紫の息（近づかない）

#### 攻撃タイミング
```
# ドラゴンが中央に降りた時が最大チャンス
1. 頭を狙って剣で攻撃
2. ブレスが来たら離れる
3. 飛び立ったら弓に切り替え
```

### ダメージ値
| 攻撃方法 | ダメージ |
|---------|---------|
| 剣（頭） | 通常の1.5倍 |
| 剣（体） | 通常 |
| 弓（飛行中） | 通常 |
| ベッド爆発 | 大ダメージ（危険） |

## 緊急対応

### 吹き飛ばされた時
```
# エンダーパールで着地
minecraft_use_item {}  # パールを投げる

# または水バケツ
minecraft_place_block { block_type: "water_bucket", ... }
```

### エンダーマンに囲まれた時
- **目を合わせない**
- カボチャヘルメット装備で無効化
- 高さ2ブロックの空間に逃げる

### HP低下時
```
minecraft_eat {}  # 金のリンゴ推奨
```

## ドラゴン撃破後

### 報酬
- **大量の経験値**（約12,000XP）
- **ドラゴンの卵**（中央の噴水上）
- **エンドゲートウェイ**（外周の島へ）

### ドラゴンの卵の回収
```
# 直接壊すとテレポートする
1. 卵の2ブロック下を掘る
2. 松明を設置
3. 卵の下のブロックを壊す
4. 卵が松明に落ちてアイテム化
```

### 帰還方法
- 中央の噴水に入る
- エンドクレジットが流れる
- オーバーワールドに帰還

## Tips

- **ベッド爆発**戦法: エンドではベッドが爆発（ハイリスク・ハイリターン）
- **スローフォーリングポーション**: 落下ダメージ無効化
- **エンダーパールは多めに**: 移動と緊急脱出に必須
- **食料は金のリンゴ推奨**: 吸収ハートが便利

## Ender Pearl Sources

### Primary: Enderman Drops
- **Recipe**: Blaze powder (1) + Ender pearl (1) → Ender eye (1)
- Needed for: ender_eye x12 to open End Portal

### Enderman Spawn Locations (Per Minecraft Wiki)
1. **Overworld**
   - Light level: 0 (complete darkness)
   - Biomes: All except Mushroom Fields and Deep Dark
   - Group size: Up to 4 (Java Edition)
   - Spawn frequency: Uncommon

2. **Nether**
   - Light level: ≤7
   - Biomes: Soul Sand Valleys (uncommon), Nether Wastes (uncommon), Warped Forests (most common)
   - Group size: 4 (Java Edition)
   - Spawn frequency: Rare

3. **The End**
   - Spawn everywhere, most common location
   - Group size: Up to 4

### Alternative Methods (if enderman hunting fails)
1. **Admin `/give` command** - Available if server allows
2. **Villager trading** - Check if any librarian villagers offer ender_pearls
3. **Preexisting inventory** - Check all accessible chests first
4. **End dimension** - Easiest place to find Endermen (they spawn very commonly there, but requires ender_eyes first - circular dependency)

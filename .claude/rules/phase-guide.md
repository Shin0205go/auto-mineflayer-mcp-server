# Phase Guide - 進行段階別戦略

8 フェーズで最終目標（エンダードラゴン討伐）に到達。

## Phase 1: 拠点確立

**目標:** 一時的な安全な基地を作る
**完了条件:** 作業台・かまど・チェスト3個・シェルター
**期間:** 1-2 日 (ゲーム時間: 20-40 ゲーム日)

### 要件
- [ ] Crafting table（作業台）配置
- [ ] Furnace（かまど）配置
- [ ] Chest x3 (材料・食料・その他)
- [ ] Shelter (敵からの保護)
  - 壁: 石以上
  - 天井: ブロック で覆い尽くし
  - ドア: 入り口 閉鎖可能

### 優先順位
1. Wood 収集（最初の 30分）
2. Wooden tools（斧・ピッケル）
3. Stone tools（より良い道具）
4. Crafting table & Furnace
5. Shelter 建設
6. Chest 配置

### チェックリスト
```
Day 1:
□ Wood x20 以上
□ Wooden pickaxe & axe
□ Stone tools 取得
□ Furnace 配置

Day 2:
□ Crafting table 配置
□ Shelter 建設開始
□ Coal/wood storage
□ Chest x3 配置

Completion:
□ Shelter が敵を防ぎ、爆発に耐える
□ Crafting/Furnace アクセス可能
□ Food ストック 20 以上
```

## Phase 2: 食料安定化

**目標:** 自動再生産できる食料源を確保
**完了条件:** 畑 OR 牧場、チェストに食料 20個以上
**期間:** 2-3 日

### 選択肢

#### Option A: 小麦畑
- Advantage: 無限増殖（種が育つ）
- Seeds 入手: 草を壊す
- 水源: 最初の 1 つで十分
- Harvest: 食料になる

#### Option B: 牧場（牛・豚）
- 初期設定: 草地で牛/豚を 4-5 頭確保
- Fence で囲む
- Breed: 小麦/キャロット で繁殖
- Harvest: 肉を焼く

### 実装順序
1. Seeds 収集（Option A の場合）
2. 畑の場所選定（Base 近く）
3. 水源確保
4. Planting & 待機（1-2 ゲーム日）
5. Harvest & 食料ストック

### チェックリスト
```
□ 食料源が確立した（畑 or 牧場）
□ 継続的に生産可能な状態
□ チェストに食料 20 個以上
□ 次のフェーズ開始可能
```

## Phase 4: 鉄装備・武装

**目標:** 全員が鉄ピッケル + 鉄剣を装備
**完了条件:** Iron tools + armor
**期間:** 3-4 日

### ステップ
1. Iron ore 採掘（石ピッケル で）
2. Furnace で精錬（Coal or wood）
3. Iron pickaxe クラフト
4. Iron sword クラフト
5. Optional: Iron armor (helmet, chestplate)

### Iron 採掘箇所
- Y=0-64（浅層）: 洞窟探索
- Y=-16～0（深層）: Staircase mining
- Min distance: 100 blocks from base

### チェックリスト
```
□ Iron ore x30 以上 採掘
□ 全員が Iron pickaxe 装備
□ 全員が Iron sword 装備
□ Iron armor（optional）
□ Diamond ore 採掘準備
```

## Phase 5: ダイヤ取得・エンチャント

**目標:** エンチャント台を設置
**完了条件:** Enchanting table 作動
**期間:** 3-4 日

### 前提
- Iron pickaxe 以上（Diamond ore 採掘に必要）
- Obsidian x14（Nether portal 用）

### ステップ
1. Diamond ore 採掘（Iron pickaxe）
   - Y=-59～-64（Deep layer）
   - Staircase mining or Branch mining
2. Diamond x3 → Pickaxe
3. Obsidian 採掘 → Nether portal 建設
4. Book & Bookshelves 用紙/本集め
5. Enchanting table クラフト

### チェックリスト
```
□ Diamond x4 以上
□ Diamond pickaxe クラフト可能
□ Obsidian x14 確保
□ Nether portal 建設可能
□ Enchanting table 配置
□ Bookshelves x15 配置
```

## Phase 6: ネザー進出・ブレイズロッド

**目標:** ブレイズロッド 7本 + エンダーパール 12個
**完了条件:** ドロップ確保
**期間:** 4-5 日

### ステップ
1. Nether portal 構築・点火
2. Nether 探索 → Fortress 発見
3. Blaze spawner 特定
4. Combat: 弓 or 剣で Blaze 倒す
5. Enderman 狩り（Ender pearl 用）

### ブレイズロッド 入手
- Blaze 倒す: ブレイズロッド ドロップ
- 敵 HP: 20 (10心)
- 攻撃方法: 弓推奨（近接は火傷リスク）
- 倒す数: 7+ 匹

### エンダーパール 入手
- Enderman 狩り
- Nether or Overworld
- ≧ 12 個確保

### チェックリスト
```
□ Nether portal 点火
□ Fortress 発見・マッピング
□ Blaze x7+ 倒す
□ ブレイズロッド x7 確保
□ エンダーパール x12 確保
□ Blaze powder クラフト
□ Ender eye x12 クラフト
```

## Phase 7: エンド要塞・ポータル準備

**目標:** End portal frame の位置を確認、起動準備
**完了条件:** 最初の Ender eye で要塞発見
**期間:** 1-2 日

### ステップ
1. Ender eye throw → 要塞方向指示
2. 複数回 throw で三角測量 → 位置確定
3. 要塞発見・探索
4. Portal room 特定
5. Eye of Ender x12 装備確認

### チェックリスト
```
□ Ender eye クラフト完了 (x12)
□ 要塞位置 特定
□ 要塞へのルート確立
□ Portal room 到達
□ 敵対 dragon detection
```

## Phase 8: ドラゴン討伐・勝利

**目標:** エンダードラゴン を倒す
**完了条件:** ドラゴン死亡、エンド要塞から脱出
**期間:** 1 日

### 戦略
1. Crystal 破壊（Healing を止める）
2. Dragon damage（弓 or 剣）
3. Healing crystal 対応
4. Phasing で移動回避
5. 敵 Enderman/Shulker 対応

### チェックリスト
```
□ 弓 & 矢（または high-level 剣）
□ Food ストック x30+
□ Blocks（防御用）
□ Portal 捜索と脱出計画
□ Victory!
```

## Multi-Bot Coordination

### リーダー (Claude1)
- フェーズ宣言: `[フェーズ] Phase N 開始`
- タスク指示: `[指示] @Claude2 Wood 採集 50個`
- 進捗監視: 定期的に `mc_status` 確認
- 完了報告: `[報告] Phase N 完了条件達成`

### フォロワー (Claude2-7)
- リーダー指示に従う
- 指示がなければ Phase 目標で自律行動
- 毎アクション報告: `[報告] Task X: progress`
- 問題発生時: `[警告] Issue description`

### チャットルール
```
[フェーズ] Phase 1 開始
  ↓ リーダーが宣言
[指示] @全員 Wood 採集開始
  ↓ フォロワー実行
[報告] Wood x20 確保、Wooden pickaxe 作成中
[報告] Phase 1 完了条件達成（作業台配置完了）
  ↓ リーダーが確認
[フェーズ] Phase 2 開始
```

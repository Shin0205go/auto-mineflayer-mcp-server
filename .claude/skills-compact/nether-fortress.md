## 見つけ方
```js
// X軸方向に移動（要塞はX軸沿いに生成）
const s = await bot.status();
await bot.moveTo(s.position.x + 100, 60, s.position.z);
// ネザーレンガの構造物を目印に繰り返す
```

## ブレイズ攻略
```js
await bot.combat("blaze"); // flee_at_hp=8がデフォルト
// 盾で火の玉を防いでから近接攻撃
// 目標: ブレイズロッド10本以上
```

## ウィザースケルトン（頭蓋骨）
- ドロップ率2.5%（低い）
- 高さ2通路に誘導して安全に攻撃
- ウィザー召喚に3個必要

## ネザーウォート
- 要塞の階段部屋（ソウルサンドの上）
- 必ず回収 → ポーション醸造の基本材料

## 必須装備
- ダイヤ/鉄防具、盾、弓+矢64、食料64、丸石128

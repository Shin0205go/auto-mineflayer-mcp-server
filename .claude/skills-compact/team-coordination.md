## 接続直後（全員）
1. `mc_chat()` — リーダー指示を確認
2. `mc_status()` — 状況把握
3. 指示あり → `mc_chat(message="[了解] ...")` + 実行
4. 指示なし → Day 1 Protocol（survival.md参照）

## リーダー（Claude1）
1. `mc_chat()` で全ボット状況を確認
2. `mc_chat(message="[フェーズ] Phase N 開始。目標: ...")` で宣言
3. 各ボットに `[指示] @ClaudeN タスク(座標含む)` を送信
4. 2アクションごとに `mc_chat()` で報告確認

## フォロワー優先順位
1. 人間チャット → 最優先
2. `[指示]` → 即実行
3. `[SOS]` → 近ければ救援
4. フェーズ目標で自律行動

## しきい値
- チェスト食料 < 20 → 空き時間に補充
- hunger < 15 → `mc_eat()` 即時
- HP < 4 → `mc_flee()` + 食べる

## チャットタグ
- `[フェーズ]` リーダーのみ、フェーズ宣言
- `[指示]` リーダーのみ、タスク割当
- `[了解]` 指示への応答
- `[報告]` 進捗・完了
- `[SOS]` 緊急救助要請
- `[資源]` 発見共有

## 禁止
- 3アクション以上チャット未確認
- リスポーンでHP回復（食料を食べろ）
- 指示なしで勝手なフェーズ行動
